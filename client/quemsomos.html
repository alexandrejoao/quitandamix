Quem Somos

Quitanda Online é uma loja virtual (e-comece) fictícia criada como base de estudo de caso para o curso "Desenvolvimento de uma Loja Virtual com Bootstrap 5 e ASP.NET 5" disponibilizando gratuitamente pelo Prof. Ricardo Maroquio em seu canal no youtube.

O código-fonte pode ser obtido via GitHub, conforme orientações apresentadas ao longo dos vídeos do curso no youtube. Este estudo de caso pode ser utilizado livremente como base para outras lojas virtuais, sem ônus para o usuário.

Bons estudos e boa programação.
